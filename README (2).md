# Guia de Instalação do Plugin "Flashcard Duplicates Finder" para RemNote

Este guia explica como instalar e usar o plugin "Flashcard Duplicates Finder" para o aplicativo RemNote. O plugin ajuda a identificar flashcards duplicados, idênticos e similares na sua base de conhecimento.

## Pré-Requisitos

- Uma conta do RemNote
- Acesso à internet

## Instruções de Instalação

### Método 1: Instalação a partir do Marketplace de Plugins (Recomendado)

1. Abra o RemNote e faça login na sua conta
2. Clique no ícone de engrenagem (⚙️) no canto inferior esquerdo da tela
3. Na barra lateral esquerda, role para baixo e clique em "Plugins"
4. Clique na aba "Explorar" (ou "Explore")
5. Na barra de busca, digite "Flashcard Duplicates Finder"
6. Quando o plugin aparecer nos resultados, clique no botão "Instalar" (ou "Install")
7. Confirme a instalação quando solicitado

O plugin agora estará instalado e pronto para uso!

### Método 2: Instalação Manual

Se o plugin não estiver disponível no Marketplace, você pode instalá-lo manualmente:

1. Baixe os arquivos do plugin (arquivo ZIP) deste repositório
2. Descompacte o arquivo ZIP em uma pasta no seu computador
3. Abra o RemNote e vá para Configurações > Plugins
4. Clique na aba "Build"
5. Clique em "Load unpacked plugin"
6. Navegue até a pasta onde você descompactou os arquivos e selecione-a
7. Clique em "Carregar plugin" (ou "Load plugin")

## Como Usar o Plugin

Depois de instalar o plugin, você pode acessá-lo das seguintes maneiras:

### Usando o Comando

1. Pressione `Ctrl+P` (ou `Cmd+P` no Mac) para abrir a paleta de comandos
2. Digite "Encontrar Duplicatas de Flashcards"
3. Pressione Enter para executar o comando

### Usando a Barra Lateral

1. Clique no ícone do plugin na barra lateral direita
2. O painel do plugin será aberto

### Principais Funcionalidades

- **Buscar Duplicatas**: Clique no botão "Buscar Duplicatas" para analisar todos os seus flashcards
- **Filtrar Resultados**: Use a caixa de texto para filtrar os resultados por palavra-chave
- **Visualizar Cartões**: Clique em "Ver" para abrir um cartão específico
- **Marcar como Verificado**: Clique em "Verificar" para marcar um cartão como verificado
- **Mesclar Cartões**: Clique em "Mesclar" para combinar dois cartões duplicados
- **Ignorar Par**: Clique em "Ignorar" para remover um par da lista de resultados

## Configurações do Plugin

Para personalizar o plugin, vá para Configurações > Plugins > Flashcard Duplicates Finder:

- **Limiar de similaridade**: Ajuste o nível de similaridade para detecção (de 0.5 a 1.0)
- **Verificar apenas o front dos flashcards**: Ative para verificar apenas o lado frontal dos cartões
- **Habilitar busca fuzzy**: Ative para usar algoritmos de busca aproximada
- **Tags da lista negra**: Liste tags separadas por vírgula para excluir cartões da análise

## Solução de Problemas

Se você encontrar problemas ao usar o plugin:

1. **O plugin não aparece**: Verifique se a instalação foi concluída com sucesso
2. **Erro ao buscar cartões**: Certifique-se de que você tem permissões de leitura para o plugin
3. **Nenhum resultado encontrado**: Ajuste o limiar de similaridade para um valor mais baixo
4. **Plugin lento**: Reduz o número de flashcards analisando apenas documentos específicos

## Informações Adicionais

Este plugin foi desenvolvido para ajudar os usuários do RemNote a manter suas bases de conhecimento organizadas e livres de duplicações. Ele usa algoritmos de similaridade de texto para identificar cartões que podem conter informações redundantes.
